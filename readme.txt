
Як працювати з шаблоном №2.
Для успішного опрацювання шаблону треба виконати дві дії:
1. Відкрити у редакторі сторінку index.html і замінти у шаблоні імена та фото на свої імена та фото(name: "Іван"; path_photo: "./photo/Ivan.png");
 {
     id: "son", name: "Son", path_photo: "./photo/son.png", link: "#sonModal"
 },
2. Відкрити у редакторі сторінку winMoodalData.js і замінити зміст усіх модальних вікон
на дані про своїх рідних.
<!--/////////////  Ivan   ///////////////////////-->
<div id = "sonModal" class="personName-eclipse" >
    <div class="personModalInfo">
<div class="titleModal">
<img src="./photo/Ivan.png" width="48">
<h2>Ivan</h2>
<a class="close" href="#">&times;</a>
</div>
        <div class="contentModal">
            <p>About Ivan</p>
        </div>
    </div>
</div> 
<!--/////////////////////////////////////////////////////-->

How to work with template #2.
To successfully process the template, you need to perform two actions:
1. Open the index.html page in the editor and replace the names and photos in the template with
their names and photos (name: "Ivan"; path_photo: "./photo/Ivan.png" )
{
    id: "son", name: "Son", path_photo: "./photo/son.png", link: "#sonModal"
},
2. Open the winMoodalData.js page in the editor and replace the content of all modal windows
on data about their relatives.
<!--/////////////  Ivan   ///////////////////////-->
<div id = "sonModal" class="personName-eclipse" >
    <div class="personModalInfo">
